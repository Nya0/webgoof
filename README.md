# webgoof
**DO NOT USE IN PRODUCTION**
simple webserver built for fun and learning.


## Features

- handles basic HTTP/1.0 requests
- serves static files
- thats about it

## Running

```bash
make run
```
test it on port 3030
```bash
curl -v http://localhost:3030/
```
